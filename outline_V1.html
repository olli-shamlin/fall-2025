Course Outline

  1. Hardware
    1. Types/kinds of devices: general vs specific purpose; desktop/personal computer, laptop, tablet, and phone
    2. Internals -- the components inside digital devices & their purpose
      1. "Brain": processor, memory, and storage
      2. Connectivity: ethernet, Wi-Fi, cellular, bluetooth
      3. User interface: screen/touchscreen, keyboard, mouse/trackpad, camera, speakers, mic
      4. Ports & Peripherals
    3. Processors, memory, & storage: what the role of each is and how each works (ie, performs its role)
    4. Size: how the power, speed, and capacity of devices is measured
    5. Common user interface gestures (eg, click/double-click/right-click, drag, select)
    6. Compare/contrast between desktop/laptop and tablet/phone (ie, mouse/touchpad vs touchscreen)
  2. Software
    1. Types/kinds: desktop, mobile, and web
    2. What are "code" and "data" and the relationship between them
    3. Understanding the size of data: bytes, MB, GB, TB, etc
    4. Operating Systems
      1. Types/kinds/brands: desktop vs mobile; macOS, Windows, ChromeOS, iOS, Android
      2. File system: files/documents, directories/folders, Finder/Navigator
      3. Settings/preferences
    5. Common functions/capabilities: file open/save, app start/stop, print, etc
    6. Common user interface components
      1. Controls: menus, toolbars, buttons, textboxes, etc
      2. Icons and there meaning (e.g., search, filter, sort, settings)
  3. Internet
    1. Computer networks (the building blocks of the internet) and how they work
    2. Hardware infrastructure: The Internet Backbone
    3. Software infrastructure: Web servers and services
    4. Use cases -- what happens "under the covers" when you do each of the following types of activities with particular emphasis on how data moves (and/or is replicated) between your device and the associated internet site/service
      1. Consuming information
        1. Browsing a page from a website
        2. Streaming: listening to music on Spotify; watching a video on YouTube
      2. Communication: email, texting, messaging
      3. Posting, sharing, uploading/downloading
      4. Signing on to a site using userid and password, MFA/TFA, passkeys, or social sign on
  4. Cloud
    1. What the cloud is (which will also answer "where the cloud is")
    2. Commercial vs consumer clouds
    3. Key cloud features (ie, why use the cloud; see use cases below)
    4. Use cases
      1. Storing data in the cloud
      2. Backing up to the cloud
      3. Syncing data across devices via the cloud
      4. Sharing files and collaborating with others using the cloud
  5. Navigation
    1. The importance of discerning which of the items each of the above surfaces is right/best for you given the context of your need/interest
    2. Search engines: (1) internet vs site (2) how they work
    3. Recommendation engines: what they are and how they work
    4. Feeds and "the Algorithm": what they are and how they work
  6. Safety
    1. Backing up your devices
    2. Encryption
      1. What it is & how it works
      2. Strong vs weak encryption
      3. Kinds of encryption: at rest, in motion, end-to-end
    3. Relative strengths of the kinds of 2FA/MFA authentication
    4. Password managers and passkeys
  7. Privacy
    1. Key point: things we do when connected to the internet create data that is a history of our activities; this data is collected/retained by the owners of the sites we use, and they can derive insights into our preference and behaviors which they can use to entice us to retain our attention/increase our use as well as drive their revenue generation (primarily through ad sales)
    2. How we are identified on the internet: sign-in ids, fingerprints & footprints
    3. What the information we provide when create an account reveals about you
    4. How things we do on social media reveals additional information about us
